尊敬的强：

您好！谨此通知您，我们已更新 Google Cloud Platform 的《数据处理和安全条款》（以下简称“DPST”），并将 DPST 更名为云端数据处理附录（以下简称“CDPA”）。

本次更新包括以下内容：

  • 在《欧盟标准合同条款》中纳入英国信息委员会新颁布的国际数据传输附录（以下简称“英国附录”），取代之前根据《数据保护指令 95/46》批准的单独的《英国标准合同条款》；
  • 将 DPST 和之前为 Google Workspace 和 Cloud Identity 提供的《数据处理修正条款》整合为一份 CDPA，以简化和加强我们在多项 Cloud 服务的数据处理方面做出的承诺。

需采取的措施

对于本次更新，您无需采取任何措施。不过，如果您的账单邮寄地址在欧洲、中东和非洲地区以外，并且您在使用 GCP 时需要遵守任何欧盟数据保护法律，那么您必须按照 Google Cloud 隐私权法规遵从和记录的说明来证明您需要遵守相应法律。

我们随时为您提供帮助

如要详细了解 Google 为确保遵守《欧盟标准合同条款》和《英国附录》而采取的措施，请参阅 Google Cloud 针对《欧盟标准合同条款》的做法白皮书。

如果您有任何问题，可以与 Google Cloud 数据保护团队联系。请在说明字段中添加此电子邮件的主题。

感谢您选择 Google Cloud Platform。
